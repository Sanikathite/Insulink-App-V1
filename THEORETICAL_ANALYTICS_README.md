Simple Keywords Guide — For Finding Research Papers Quickly

Purpose:
Short, easy keyword lists and search suggestions for each analytics topic. Copy these into Google Scholar, IEEE Xplore, or ACM Digital Library.

Alarm Frequency
- Keywords: alarm frequency, event rate, Poisson process, non-homogeneous Poisson
- Short queries: "alarm frequency Poisson"; "event rate estimation alarms"; "non-homogeneous Poisson alarms"

MTTA / MTTR (Time-to-Event)
- Keywords: mean time to acknowledge, mean time to resolve, MTTA, MTTR, survival analysis, censored data
- Short queries: "MTTR survival analysis"; "mean time to acknowledge censored"

Repeated Alarms / Burst Detection
- Keywords: repeated alarms, burst detection, burstiness, Fano factor, Markov-modulated Poisson
- Short queries: "burst detection alarms"; "repeated alarms Fano factor"

Anomaly Detection (Spikes & Patterns)
- Keywords: anomaly detection, change-point detection, CUSUM, EWMA, Isolation Forest, LSTM anomaly
- Short queries: "alarm anomaly detection CUSUM"; "Bayesian online change point alarms"

Device Health Scoring
- Keywords: device health score, health index, multi-criteria decision, AHP, TOPSIS, predictive maintenance
- Short queries: "device health scoring IoT"; "health index predictive maintenance"

Alarm Correlation & Event Modeling
- Keywords: alarm correlation, multivariate point process, Hawkes process, causal inference alarms
- Short queries: "Hawkes process alarms"; "alarm correlation multivariate point process"

Data & Preprocessing Keywords
- Keywords: event logs, alarm log, action logs, telemetry, deduplication, time alignment
- Short queries: "alarm log preprocessing"; "event log deduplication"

Evaluation & Metrics
- Keywords: precision recall detection latency, ROC-AUC, PR-AUC, C-index, survival concordance
- Short queries: "alarm detection precision recall"; "MTTR concordance index"

Practical Search Tips
- Combine domain + method: e.g., "SCADA anomaly detection Isolation Forest" or "IoT MTTR survival analysis".
- Use review/survey filters: add "survey" or "review" to find overviews.
- Try year filters to get recent work (e.g., 2020..2026).
